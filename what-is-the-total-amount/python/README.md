# Requirement

- [Python 3.9](https://www.python.org/)
- [Visual Studio Code](https://code.visualstudio.com/download)

## Libs

- aiohttp
- requests

<br>

# Installation

- Install Python from link above
- Install IDE Vs Code or other
- Terminal Way:
  - run `pip install -r requirements.txt`
  - run `python main.py`
